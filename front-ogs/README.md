### 1. Clone repository

### 2. Installation :

Front-End (VueJS 2.6.11) :

-   Install Node modules :
    ```
    npm install
    ```
-   Serve (the project is getting built by Webpack and then served via a **local http-server**) :
    ```
    npm run serve
    ```

### Compiles and minifies for production
